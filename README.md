# TypeScritp X Express.js で作るサーバサイド開発基盤

### 入っているもの
- Express.js
- TypeScript
- ESLint
- Prettier
- node-ts
- nodemon
- Docker

### 使い方
$ git clone https://github.com/cdlscdlscdls/tsx.git

$ cd tsx

$ docker-compose up --build web

=> http://localhost:3000
